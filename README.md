# Палиндаты
Простое приложение для поиска зеркальных *(палиндромных)* дат.
**Палиндромная дата** — это такая дата, которая читается одинаково слева направо и справа налево *(например, 01.02.2010)*.

## Описание
Это простое приложение находит палиндромные даты в заданном диапазоне лет. Вводятся два числа: начальный год и конечный. После нажатия на кнопку "Посчитать", программа сама найдёт все палиндромные даты между введёнными годами. Ищет даты от 0-го года до 9999-го включительно. Также указывается суммароное кол-во палиндромных-дат, и даты можно скопировать.
